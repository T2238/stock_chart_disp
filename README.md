# 株価チャート表示アプリ

東証上場銘柄のローソク足チャートをブラウザで表示するWebアプリです。  
スマートフォンを含む任意のデバイスからアクセスできます。

---

## 機能一覧

| 機能 | 詳細 |
|------|------|
| ローソク足チャート | 日足・週足・月足、上場来全データを取得 |
| テクニカル指標 | 移動平均(MA5/20/75)・ボリンジャーバンド・RSI・MACD |
| 複数銘柄比較 | リターン率(%)で起点を揃えて重ね表示（最大3銘柄） |
| 指数比較 | 日経225・TOPIX ETF・S&P500・NASDAQ・NYダウ・ドル円 |
| 業種別ETF比較 | 所属業種に対応するNEXT FUNDS業種ETF(1615〜1631)と比較 |
| 規模別ETF比較 | TOPIX Core30 ETF(1311.T)と比較（Core30銘柄のみ） |
| JPXプライム150 | 構成銘柄に青バッジ表示・ETF(2080.T)と比較可能 |
| JPX日経400 | 構成銘柄に緑バッジ表示・ETF(1592.T)と比較可能 |
| 時価総額表示 | 現在値 × 発行済株式数（億円／兆円） |
| 市場フィルタ | プライム・スタンダード・グロースで絞り込み |

---

## データソース

| データ | 取得元 | 更新頻度 |
|--------|--------|----------|
| 株価（OHLCV） | Yahoo Finance (yfinance) | 5分キャッシュ |
| 東証銘柄リスト | J-Quants API V2 | 1時間キャッシュ |
| 発行済株式数 | Yahoo Finance (yfinance) | 1時間キャッシュ |
| JPXプライム150構成銘柄 | JPX公式CSV | 24時間キャッシュ |
| JPX日経400構成銘柄 | JPX公式CSV | 24時間キャッシュ |

## ローカル環境でのセットアップ

### 必要なもの
- Python 3.10 以上
- J-Quants API キー（無料）: https://jpx-jquants.com

### 手順

```bash
# 1. リポジトリをクローン
git clone https://github.com/T2238/stock_chart_disp.git
cd stock_chart_disp

# 2. 依存パッケージをインストール
pip install -r requirements.txt

# 3. 環境変数を設定
# .env ファイルを作成して API キーを記入
echo JQUANTS_API_KEY=ここにAPIキー > .env

# 4. アプリを起動
python -m streamlit run app.py
```

ブラウザで http://localhost:8501 が開きます。

## Streamlit Community Cloud へのデプロイ

スマートフォンなど外部からアクセスできる URL を発行できます（無料）。

### 手順

1. [share.streamlit.io](https://share.streamlit.io) を開き、GitHub アカウントでログイン
2. **New app** をクリック
3. 以下を設定：
   - Repository: `T2238/stock_chart_disp`
   - Branch: `master`
   - Main file path: `app.py`
4. **Advanced settings → Secrets** に以下を貼り付け：
   ```toml
   JQUANTS_API_KEY = "ここにAPIキーを入力"
   ```
5. **Deploy!** をクリック

数分後に `https://xxxxx.streamlit.app` の URL が発行されます。  
以降は `master` ブランチへの push で自動的に再デプロイされます。

## 使い方

### 銘柄を選ぶ

サイドバー上部の **市場フィルタ** でプライム／スタンダード／グロースを絞り込み、  
セレクトボックスに銘柄名またはコードを入力して選択します。

### チャートを見る

- チャート内の **1M / 3M / 6M / 1Y / 3Y / 5Y / 10Y / Max** ボタンで表示期間を切り替え
- サイドバーの **初期表示期間** スライダーで、銘柄切替時のデフォルト期間を設定
- **足種**（日足・週足・月足）を切り替え可能
- チャートはドラッグでパン、スクロールでズーム操作できます

### テクニカル指標を表示する

サイドバー下部のチェックボックスで各指標をオン／オフします。

| 指標 | 説明 |
|------|------|
| 移動平均 MA(5/20/75) | 短期・中期・長期のトレンドを確認 |
| ボリンジャーバンド (±2σ) | 価格の標準的な変動範囲 |
| RSI (14) | 70以上で買われすぎ、30以下で売られすぎ |
| MACD (12,26,9) | トレンドの転換点を確認 |

### 比較モードを使う

サイドバーの **比較モード** トグルをオンにすると、起点を 0% として正規化したリターン率チャートに切り替わります。

- **比較銘柄**：最大3銘柄まで追加可能
- **指数比較**：日経225・S&P500 等と重ねて表示
- **業種別ETF**：選択銘柄の業種に対応するETFが自動で選択肢に表示
- **JPXプライム150 / JPX日経400 ETF**：構成銘柄の場合のみ表示

### 指数組入バッジ

チャート下部の銘柄情報欄に、以下のバッジが表示されます。

- 🔵 **JPXプライム150** … JPXプライム150指数の構成銘柄
- 🟢 **JPX日経400** … JPX日経400指数の構成銘柄

## ファイル構成

```
stock_chart_disp/
├── app.py              # Streamlit メインアプリ
├── chart.py            # Plotly チャート生成
├── jquants.py          # J-Quants API 銘柄リスト取得
├── index_members.py    # JPXプライム150・JPX日経400 構成銘柄取得
├── indicators.py       # テクニカル指標計算
├── config.py           # 設定（指数・ETFマッピング等）
├── requirements.txt    # 依存パッケージ
├── .env                # APIキー（ローカル用・Git管理外）
└── .streamlit/
    └── secrets.toml.example  # Streamlit Cloud 用シークレットテンプレート
```

## 注意事項

- 株価データは Yahoo Finance から取得しており、**リアルタイムではなく 15〜20 分の遅延**があります
- J-Quants API キーの無料プランは取得できるデータに制限があります
- JPXプライム150・JPX日経400 の構成銘柄リストは定期選定（年1回程度）で変更されます。アプリは 24 時間ごとに JPX 公式 CSV から自動取得します
- 本アプリは情報提供を目的としており、投資判断の根拠とすることを推奨するものではありません
